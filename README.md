# Image Processing Assignment

## Overview

This assignment demonstrates basic image processing techniques using Python, including:

* Harris Corner Detection
* Otsu Thresholding

---

## Task 1: Harris Corner Detection

Applied Harris Corner Detection to identify key points (corners) in an image.

**Output:**

* Image with detected corners highlighted

---

## Task 2: Otsu Thresholding

Applied Otsu’s method to automatically determine the optimal threshold for image segmentation.

**Outputs:**

* Original Image
* Image Histogram
* Thresholded Image

---

* OpenCV
